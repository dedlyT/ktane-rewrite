# KTANE
**The project files for the real life keep talking and nobody explodes replica I'm working on.** I've been working on this project on-and-off for the last few years, just as a project to fill time when I'm bored. This is my 4th rewrite.

## Physical Hardware
Comprises of a Raspberry Pi master and several Raspberry Pi PICO slaves, who communicate to each other in a UART daisychain. The master is the "Timer" module, as no bomb can exist without a timer. The slaves will be the other modules. I'll supply pictures as I rewire the modules, and 3D-model/3D-print the containers.
